Troubleshooting

I can't connect with my web browser!
Maybe you have an HTTP proxy set up. Try referring to your machine by its name instead of using localhost.
It crashes!
Yuck. I can send you a version with extra information with debugging information turned on. When you get a crash, click on "debugger" in the crash window, type "sc" then enter in the debugger window, and e-mail me what you get back. Make sure to tell me anything you were doing at the time.
Clicking rewind makes the player freak out!
In particular, it will try to rewind at -18000% and the player won't be able to keep up. This is a bug in SoundPlay versions 4.3 and before. Check for the latest release of SoundPlay for a fix.
Nothing's showing up!
Did you run the installer and let it build mp3 indexes on your volumes?
How do I turn on looping?
Turn on the WinAMP user interface, click on the little looping button, then go back to the nice BeOS interface.
It quits when I empty all the playlists!
This is the normal behaviour when you are running the WinAMP skin. I can't prevent it. The only solution is to either never empty the playlists OR don't use the WinAMP skin.

Getting and Giving Help

The Be in your Stereo home page has a number of useful resources:
  • Software updates.
  • A copy of this manual where users can leave comments, tips, and suggestions for each other.
  • A mailing list where users can get involved in more in-depth discussions.
You may also wish to try the newsgroup comp.sys.be.misc, or the Be-run mailing list Be User Talk.

Please try these resources first before e-mailing me for help.

Contributions to the software, including bug reports and feature requests, are particularly welcome.
